# Mensagens de boas-vindas para jogadores específicos
# Formato: NomeDoJogador:Mensagem
# Uma linha por jogador, linhas iniciadas com # são comentários

# Exemplo:
# Steve:Bem vindo Steve! Esta é uma mensagem automática, peço que verifique seu microfone e configure o chat de voz. Obrigado e boa jogatina!
# Alex:Olá Alex! Seja bem-vindo ao servidor. Não esqueça de configurar o voice chat!

# Adicione seus jogadores aqui:
SeuNomeDeUsuario:Bem vindo! Esta é uma mensagem automática, peço que verifique seu microfone e configure o chat de voz. Obrigado e boa jogatina!
